   - This sets your status to "Busy"
   - Your status note will be displayed on the public dashboard
   - If you set a duration, a countdown timer will appear

## â“ Troubleshooting

### "I don't see the Employee Control Panel"
- Make sure you logged in with an employee account
- Check that your role is "employee" (not "admin")
- Try logging out and logging back in

### "Status changes aren't showing on the dashboard"
- The changes should appear instantly (real-time updates)
- Try refreshing the public dashboard page
- Check your internet connection

### "The timer isn't working"
- Make sure you selected a duration (15m, 30m, or 60m)
- The timer only appears when you're busy
- It will automatically expire when time runs out

### "I can't update my status"
- Make sure you're logged in
- Check that you have the correct permissions
- Try logging out and back in
